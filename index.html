Introducción

Una página web es un documento digital que puede visualizarse desde un navegador. Su creación implica el uso de lenguajes como HTML (estructura), CSS (diseño) y JavaScript (interactividad).

Este sitio te guiará paso a paso para entender cómo construir una página web académica e interactiva desde cero.

Estructura básica de una web

  1. HTML: define el contenido y la estructura.
  2. CSS: aplica colores, fuentes y disposición visual.
  3. JavaScript: añade dinamismo y control de eventos.

Herramientas necesarias

  • Editor de texto (Visual Studio Code o Notepad++)
  • Navegador web (Google Chrome, Firefox, Edge, etc.)
  • Cuenta en GitHub para publicar tu proyecto

Publicación en línea

Una vez creada tu página, puedes publicarla mediante GitHub Pages. Este servicio gratuito permite mostrar tu sitio al mundo a través de una dirección web única.

Para activar GitHub Pages, ve a la pestaña Settings → Pages y selecciona la rama “main”. Luego, guarda los cambios y espera unos segundos para obtener tu enlace público.
